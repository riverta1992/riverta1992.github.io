      LeniaにおけるFFTとShiftの役割 #

      1.前提知識 #

      \(\) \(N\times N\) の行列 \(A\)、\(K\)について考えてみます。\(A\) は Lenia ワールドの現在の状態を表し、各セルは0から1の値をとります。\(K\) は畳み込み演算を行うための行列で、以下のように正規化されていると仮定します。

      \[\sum_{i=0}^{N-1}\sum_{j=0}^{N-1}K(i,j) = 1\]

      畳み込み定理より、以下の公式が成り立ちます。 \[\mathcal{F}[A*K] = \mathcal{F}[A]\cdot\mathcal{F}[K]\]

      つまり、畳み込み \(A*K\) を計算するにはフーリエ変換と逆フーリエを駆使して \(\mathcal{F}^{-1}[\mathcal{F}[A]\cdot\mathcal{F}[K]]\) を計算すればよいことになります。一見して面倒な処理に思えますが、計算量という観点でメリットがあります。
