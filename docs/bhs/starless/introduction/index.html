      1.趣旨 #

      Riccardo Antonelli による Python 製のブラックホールレイトレーサーを導入します。プログラムを動かすにあたって必要な修正について述べ、最後にデフォルトパラメータを用いて実際に画像を生成するところまで行います。本記事においては細かい設定や具体的なアルゴリズムについては触れませんので、ご了承ください。

      3.エラー #

      Github からソースコードをダウンロードします。

      $ git clone https://github.com/rantonels/starless.git
$ cd starless

      starlessのwikiにしたがい、、プログラムに以下のオプションを与え、実行します。
